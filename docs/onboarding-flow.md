## Onboarding & Profiling Flow

1. User selects preferred language and accepts terms.
2. User creates an account (email + password).
3. User enters age and basic demographic details.
4. User selects skin type (Oily, Dry, Combination).
5. User selects primary concerns (e.g. Acne, Wrinkles, Sensitivity).
6. User selects budget range.
7. App generates an initial routine and product recommendations.
