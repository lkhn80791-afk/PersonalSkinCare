# PersonalSkin Monorepo

PersonalSkin is a personalized skincare application that helps users build and track morning and evening routines based on their unique skin profile.

## Structure

- `src/backend`: Node.js + TypeScript API (Express + Sequelize + PostgreSQL)
- `src/mobile`: Flutter mobile app (iOS & Android)
- `docs`: Architecture and product documentation
- `tests`: Backend and mobile tests
- `resources`: Shared assets and configuration

## Development

### Backend

```bash
npm install
npm run dev
```

### Mobile

Open `src/mobile` in your Flutter environment and run:

```bash
flutter pub get
flutter run
```
